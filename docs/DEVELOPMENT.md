## Архитектура системы

```
cryptocore.py — Точка входа CLI

src/cli_parser.py — Парсер аргументов командной строки

src/crypto_core.py — Основной API для шифрования/дешифрования

src/modes/ — Реализации режимов AES
```

## Тестирование

### Запуск тестов

#### Все тесты

```bash

python -m pytest tests/ -v
```

#### Быстрые тесты

```bash

python -m pytest tests/ -q
```

#### Конкретный тест

```bash

python -m pytest tests/test_mac.py -v
```
